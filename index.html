C++ vtable Error

Posted on 2019-09-02 Edited on 2022-03-13 In c++

解决方法： 1. 对于纯虚函数需要加上函数体 {}，即 virtual void example(); 要改为 virtual void example() {} 2. 含有虚函数的类的析构函数要声明为virtual。
